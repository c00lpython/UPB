# UPB Parser — Universal Parser Builder

**UPB Parser** — это локальный конструктор парсеров с поддержкой CloakBrowser, Telegram-бота и персональных данных.

## 🚀 Возможности

- 🥷 **CloakBrowser** — обход антибот-защиты (58 C++ патчей)
- 🤖 **Telegram-бот** — регистрация, логин, просмотр и скачивание результатов
- 👤 **Персональные папки** — каждый пользователь видит только свои данные
- 📊 **Plotly графики** — автоматическая визуализация результатов
- 📜 **Скролл до конца** — подгрузка всех элементов на странице
